# Hey Artcraft Web

Plataforma web para la gestión y promoción del emprendimiento costarricense **Hey Artcraft**, especializado en artículos personalizados de madera y acrílico.

---

## Objetivo

Desarrollar una plataforma web que fortalezca la presencia digital de Hey Artcraft, permitiendo a sus clientes explorar el catálogo, personalizar productos y realizar pedidos de forma autónoma, mientras el equipo administrativo gestiona eficientemente el inventario, los pedidos y el historial — todo sin perder la identidad creativa y cercana del emprendimiento.

## Pasos preliminares de ejecución

### Prerrequisitos

Asegurarse de tener instalado:
- [Java JDK](https://www.oracle.com/java/technologies/downloads/) v17 o superior
- [Maven](https://maven.apache.org/) o el wrapper incluido en el proyecto (`mvnw`)
- Cuenta en [Firebase](https://firebase.google.com/) con un proyecto configurado
- Git
- Navegador web moderno

---

### 1. Clonar el repositorio

```bash
git clone https://github.com/MartiRoll29/SC-403-HeyArtcraftWeb.git
cd SC-403-HeyArtcraftWeb
```

### 2. Configurar Firebase

- Crear un proyecto en [Firebase Console](https://console.firebase.google.com/)
- Copiar las credenciales del proyecto en el archivo de configuración correspondiente del repositorio
